QCM – Naturalisation
Deux modes : Révision (réponse) • QCM (choix + score si options renseignées)

—

Filtres
Les questions peuvent ne pas avoir d’options. Dans ce cas, le mode QCM affiche “Options non renseignées”.
Import / Export (JSON)
Tu peux compléter les champs options (A-D) et answerKey pour activer le mode QCM.
Exemple :
"options":[{"key":"A","text":"..."},{"key":"B","text":"..."},{"key":"C","text":"..."},{"key":"D","text":"..."}], "answerKey":"B"
